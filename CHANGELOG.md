# Changelog

## 1.0.0

- Added the Agronomy xyloschemer category for supported Harvestry plants.
- Added ingredient-gated cultivation using each plant's harvested item.
- Added persistent regrowth based on H Value, with a minimum growth time of one day.
- Preserved active growth deadlines when unripe status is reasserted.
- Stopped ripe cultivated plants from requesting redundant turn ticks.
- Added an MIT license.
